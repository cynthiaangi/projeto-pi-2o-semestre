# 💉 Conscientização da Vacinação no Estado de São Paulo

Sistema desenvolvido durante o Projeto de Pesquisa e Inovação da SPTech com o objetivo de apoiar a análise da cobertura vacinal no Estado de São Paulo por meio da visualização de indicadores e dashboards interativos.

A aplicação foi desenvolvida para transformar dados em informações estratégicas, auxiliando na identificação de regiões com baixa adesão à vacinação e apoiando a tomada de decisões voltadas à saúde pública.

---

# 📖 Sobre o projeto

O Dashboard de Cobertura Vacinal foi desenvolvido para facilitar a análise de dados relacionados às campanhas de vacinação.

A solução permite visualizar indicadores, acompanhar tendências e consultar informações de forma intuitiva através de dashboards interativos.

Durante o desenvolvimento foram aplicados conceitos de desenvolvimento Full Stack, integração entre Front-end e Back-end, modelagem de banco de dados e visualização de dados.

---

# 🎯 Objetivos

- Disponibilizar indicadores sobre cobertura vacinal.
- Facilitar a visualização de dados por meio de dashboards.
- Auxiliar na identificação de regiões com baixa adesão às vacinas.
- Apoiar decisões baseadas em dados.
- Aplicar conceitos de desenvolvimento web e análise de dados.

---

# 🚀 Funcionalidades

- Dashboard com indicadores de cobertura vacinal
- Visualização de gráficos e métricas
- Consulta de informações por município
- Relatórios para apoio à análise de dados
- Integração entre Front-end e Back-end
- Consumo de APIs para obtenção dos dados
